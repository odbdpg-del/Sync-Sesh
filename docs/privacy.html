Sync Sesh Privacy Policy

Last updated: April 19, 2026

This Privacy Policy explains the basic information handling practices for Sync Sesh, a Discord Activity for shared countdown sessions.

Information Processed

Sync Sesh may process limited activity-related information such as:

  • Discord user identifiers needed to display participants in a session
  • Display names and avatar references provided by Discord
  • Session state such as ready status, spectator status, and timer settings

How Information Is Used

This information is used only to operate the shared countdown experience and display session participants.

Storage

For the current MVP, session state may be stored temporarily in memory for live sync. Local testing builds may also store lightweight profile data in browser local storage.

Sharing

Sync Sesh does not sell personal information. Information visible in a session is shared only with users in that session as needed for the activity to function.

Your Choices

You may stop using the service at any time. If local browser data is used during testing, you can clear it through your browser storage settings.
